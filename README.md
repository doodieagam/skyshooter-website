# 🚁 Skyshooter Production Website

Website resmi untuk **Skyshooter Production**, penyedia jasa dokumentasi **foto, video, dan drone** untuk berbagai kebutuhan seperti event, wedding, dan promosi bisnis.

---

## 🌐 Live Website

🔗 https://skyshooter.co.id/

---

## 📌 Deskripsi Project

Project ini merupakan website company profile yang bertujuan untuk menampilkan layanan, portfolio, serta memudahkan calon client dalam menghubungi Skyshooter Production.

Website ini dikembangkan dengan fokus pada:

* Tampilan visual yang menarik (visual-driven)
* Kemudahan akses informasi layanan
* Optimalisasi pengalaman pengguna (UX)

---

## 🎯 Fitur Utama

* 📸 Showcase layanan fotografi & videografi
* 🚁 Layanan drone (aerial shooting)
* 🖼️ Portfolio project
* 📱 Responsive design (mobile-friendly)
* 📞 Call to Action (kontak langsung ke client)

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* Bootstrap

---

## ⚙️ Cara Menjalankan Project (Local)

1. Clone repository:

   ```bash
   git clone https://github.com/doodieagam/skyshooter-website.git
   ```

2. Masuk ke folder project:

   ```bash
   cd skyshooter-website
   ```

3. Buka file `index.html` di browser

## 📊 Fokus Pengembangan

Beberapa hal yang menjadi fokus dalam pengembangan project ini:

* Optimasi performa website (PageSpeed & Lighthouse)
* Kompresi gambar untuk meningkatkan kecepatan loading
* Struktur UI/UX yang sederhana namun efektif
* SEO dasar untuk meningkatkan visibilitas di Google

---

## 🧪 Testing & Quality Assurance

Pengujian dilakukan secara manual dengan skenario berikut:

* Validasi tampilan di berbagai ukuran layar (responsive testing)
* Pengujian navigasi antar halaman
* Validasi link & tombol CTA
* Uji performa menggunakan Google Lighthouse & PageSpeed Insights
